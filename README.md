# Course_Election_for_SUFE
上海财经大学自动选课脚本 支持二轮三轮及开学选课

## Requirements

- Chrome/Firefox 

### Guide

1. 在门户选课网页输入自己要选的课程序号
2. 浏览器快捷键 F12 并进入 Console
3. Console 中粘贴 `main.js` 中代码, 回车运行
